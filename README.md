# Проект III. Интерактивная карта религиозных объектов Ярославля

**Карта онлайн:** https://kristinalantsova.github.io/

## О проекте

Ярославль называют "городом тысячи церквей" за его колоритные храмы, монастыри, мечети, синагоги и часовни, отражающие многоконфессиональный характер города. Особое внимание уделено пространственному распределению религиозных объектов и их концентрации в разных районах Ярославля.

## Что изображено на карте

На карте представлены границы районов Ярославля,плотность жилой застройки, религиозные объекты и распределение жителей на один религиозный объек и по районам. 

## Источники данных

Основным источником данных является OSM, однако 2/3 религиозных объектов отсутствовали в OSM, в связи с тем данные дополнялись вручную с использованием сайта "Соборы.ру".
